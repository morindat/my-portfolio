## 🌌 Overview

A futuristic, sci-fi inspired portfolio website showcasing my journey as a developer. Built with React and TailwindCSS, featuring smooth animations, interactive elements, and a purple/black aesthetic that's out of this world.

### ✨ Features That Make It Special

- 🚀 **Interactive Typewriter Effect** - Dynamic text animation in hero section
- 🎨 **Sci-Fi Aesthetic** - Custom gradient designs with purple/black theme
- 📱 **Fully Responsive** - Perfect on desktop, tablet, and mobile
- ⚡ **Smooth Animations** - Scroll effects, hover states, and transitions
- 📬 **Functional Contact Form** - With copy-to-clipboard functionality
- 🎯 **Project Showcase** - Clean cards with live demo and GitHub links
- 🌟 **Mars Project Highlight** - Unique astronomy data visualization project
- 🔗 **Social Integration** - All important links in one place

## 🏃‍♂️ Launch Locally

Want to run this on your own machine? Here's how:

```bash
# Clone this repository
git clone https://github.com/Papadizzo/portfolio.git

# Enter the project
cd portfolio

# Install dependencies (might take a lightyear ⏳)
npm install

# Launch development server
npm run dev

# Open your browser and visit
# http://localhost:5173
